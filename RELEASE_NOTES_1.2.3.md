# WitNote v1.2.3 发布简介

我们很高兴地发布 WitNote v1.2.3！这是一个重要的更新，专注于提升用户体验、修复已知问题并优化了底层的 AI 引擎集成。

## 🌟 主要更新

- **UI 界面优化**：全面优化了对话框样式，统一了视觉体验，设置面板重新布局，新增“关于软件”和“快捷键说明”。
- **Markdown 编辑器增强**：改进了编辑体验。
- **本地化修复**：修复了多语言环境下的菜单和提示词显示问题（支持繁体中文更完善）。
- **流程优化**：移除了内置的 Ollama，改为更灵活的外部调用方式，并在设置中明确了开源声明。
- **构建升级**：优化了 macOS DMG 安装包的构建流程，包含精美的背景图和图标调整。

## 🛠️ 修复与改进

- 修复了 WebLLM 初始化时可能出现的模型加载错误。
- 修复了缓存清理逻辑，确保模型能够彻底重置。
- 修复了设置面板中部分按钮状态显示不正确的问题。

感谢您的支持与反馈！由于采用了新的签名验证，首次运行时 macOS 可能会提示检查，请放心打开。
