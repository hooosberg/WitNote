## ğŸŒŸ Filosofi Inti

- **Cerdas (Smart)**: Tiga Mesin dalam Satu, Kebebasan Memilih.
  - **WebLLM**: Model ringan, perlu diunduh saat pertama kali dijalankan, lalu bekerja offline.

  - **Cloud API**: Terhubung ke kecerdasan cloud, kemungkinan tak terbatas.

  - Manajemen kartu gaya iOS, seret untuk mengatur.
  - Mode fokus cerdas â€” jendela menyempit, editor menyederhana.
- **Aman (Secure)**: Kedaulatan data.

> âŒ **Catatan Penting untuk Mac Intel**: 
> 
> Aplikasi ini **tidak mendukung** komputer Mac dengan chip Intel. Bahkan jika dipaksa berjalan, pengalamannya akan sangat buruk karena alasan berikut:
> 1. **Ketidakcocokan Arsitektur**: Mesin inferensi lokal bawaan (WebLLM/Ollama) sangat bergantung pada arsitektur ARM64 dan akselerasi perangkat keras NPU/Metal Apple Silicon.
> 2. **Kurangnya Akselerasi Perangkat Keras**: Mac Intel tidak memiliki Arsitektur Memori Terpadu. Menjalankan model terkuantisasi sangat lambat (membuat satu token bisa memakan waktu beberapa detik) dan menyebabkan perangkat menjadi sangat panas.
> 3. **Kompromi Arsitektur**: Untuk memastikan pengalaman terbaik dan ukuran paket minimum, kami telah menghapus dukungan untuk arsitektur x86_64.
> 
> Kami sangat menyarankan menggunakan perangkat Mac yang dilengkapi dengan chip Apple Silicon (seri M).

## ğŸ”§ Info Mesin AI

### 1. WebLLM (Ringan)
Aplikasi ini mencakup mesin WebLLM bawaan dengan model `qwen2.5:0.5b` (hanya macOS, pengguna Windows disarankan untuk menggunakan Ollama).
- **Kelebihan**: Tidak perlu instalasi perangkat lunak tambahan, bekerja sepenuhnya offline setelah unduhan model awal.
- **Terbaik untuk**: Tanya Jawab Cepat, pemolesan teks sederhana, perangkat low-end.

### 2. Ollama (Kekuatan Lokal)
Mendukung koneksi ke layanan Ollama yang berjalan secara lokal.
- **Kelebihan**: Menjalankan model 7B, 14B atau bahkan lebih besar, performa kuat, sepenuhnya offline.
- **Penggunaan**: Instal [Ollama](https://ollama.com) terlebih dahulu, lalu unduh lebih banyak model di Pengaturan (mis. qwen2.5:7b, llama3, dll.).

### 3. Cloud API (Cloud Tanpa Batas)
Mendukung koneksi ke Cloud API yang kompatibel dengan OpenAI.
- **Kelebihan**: Akses model paling kuat di Bumi hanya dengan satu kunci API.
- **Terbaik untuk**: Penalaran logis tingkat atas, atau ketika perangkat keras lokal tidak dapat mendukung model besar.
- **Konfigurasi**: Masukkan URL API dan Kunci di Pengaturan (Mendukung OpenAI, Gemini, DeepSeek, Moonshot, dll.).
