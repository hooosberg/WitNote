> **🎉 Mac App Storeで公開されました！**
> 
> WitNoteがMac App Storeに正式に登場しました。サポートありがとうございます！皆様のStar ⭐️が私たちが前進し続ける最大の原動力です！

**WitNote**は、macOS、Windows、Linux向けのローカルファーストAIライティングコンパニオンです。
**Ollama / WebLLM / Cloud API** の3つのエンジンを自由に切り替えることができ、極めてミニマルな**ネイティブカードインターフェース**を備え、すぐに使用できます。モデルのダウンロード以外にクラウドへの依存はなく、プライバシーの懸念もありません。知性を軽量化しました。

![Local Offline AI Note](../src/pic/witnote%20宣传截图/English/本地离线AI记事本.jpg)

## 🌟 コアフィロソフィー

- **Smart (スマート)**: 3つのエンジンを1つに、自由に選択。
  - **WebLLM**: 軽量モデル、初回実行時にダウンロードが必要ですが、その後はオフラインで動作します。
  - **Ollama**: パワフルなローカルモデル、強力なパフォーマンス、完全オフライン。
  - **Cloud API**: クラウドインテリジェンスに接続し、無限の可能性を。
- **Simple (シンプル)**: 複雑さを排除。
  - iOSスタイルのカード管理、ドラッグ＆ドロップで整理。
  - スマートフォーカスモード — ウィンドウを狭めるとエディタがシンプルになります。
- **Secure (セキュア)**: データ主権。
  - [**プライバシーポリシー**](PRIVACY.md): 100%ローカルストレージ。あなたの思考はあなただけのものです。

## ✨ 機能 (v1.3.3)

### 🆕 v1.3.3の新機能

- 🪟 **すりガラスUI** — 真新しいビジュアルデザイン、全体的にモダンなすりガラス効果
- 📐 **柔軟な3ペインレイアウト** — サイドバー、エディタ、AIパネルは個別にサイズ変更や終了が可能
- ✋ **2ペインドラッグ＆編集** — 分割ビューで仕切りを自由にドラッグして、エディタ/プレビューの比率を調整
- 📂 **Finderスタイルのファイルツリー** — カラータグ、ドラッグ移動、ホバー展開、完全にアップグレードされたコンテキストメニュー
- 📄 **マルチフォーマット対応** — PDF/Wordプレビューを追加、一般的な画像形式をサポート
  - 編集可能: `.md` `.txt`
  - 読み取り専用プレビュー: `.pdf` `.docx`
  - 画像ビューア: `.jpg` `.png` `.gif` `.webp`
- 🚀 **高速起動** — スプラッシュ画面を最適化、空白画面の待機時間を大幅に短縮
- 🔒 **App Storeリリース** — Apple Sandboxセキュリティ基準に完全準拠、Mac App Storeで入手可能に

### 🔧 主な機能

- 📝 **ピュアローカルノート** — 任意のフォルダをノート保管庫として選択、`.txt`と`.md`をサポート
- 🤖 **3-in-1エンジン** — **WebLLM** (軽量)、**Ollama** (ローカルパワー)、**Cloud API** (カスタム接続) を自由に切り替え
- ✨ **スマートオートコンプリート** — 3つのプリセット (Lite/Standard/Full)、Tabキーを押して文ごとに提案を受け入れ
- 🎭 **リッチなロールライブラリ** — 10種類以上の厳選されたロールプロンプト (ライター、翻訳者、推敲者など) を内蔵、ワンクリック切り替え＆カスタム対応
- 🌍 **グローバルコミュニケーション** — **8言語をサポート**: 
  - English, 简体中文, 繁體中文, 日本語, 한국어, Français, Deutsch, Español
  - インターフェースとAIの応答は自動的にあなたの言語に適応します
- 🔒 **プライバシーファースト** — すべてのAI推論はローカルで行われます (ローカルエンジン使用時)、データアップロードなし、Apple公証済み
- 💬 **高度なカスタマイズ** — システムプロンプトを自由に編集して、あなた専用のAIアシスタントを作成
- 🎨 **複数のテーマ** — ライト / ダーク / 禅ティー、最適化されたダークモード
- 🗂️ **カードグリッドビュー** — ドラッグ＆ドロップで並べ替え可能なiOSスタイル、洗練されたコンテキストメニュー
- 🔍 **コンテキスト認識** — AIが現在の記事やフォルダの内容を直接読み取ることができます
- 🎯 **フォーカスモード** — ウィンドウが狭くなると自動的に邪魔が入らない編集モードに切り替わります

## 💻 システム要件

### 🍎 macOS

| 項目 | 最小 | 推奨 |
|------|---------|-------------|
| OSバージョン | macOS 12.0+ | macOS 13.0+ |
| チップ | **非対応 (Intelチップ)** | **Apple Silicon (M1/M2/M3/M4/M5)** |
| RAM | - | 16GB+ |
|ストレージ | - | SSD, 4GB以上の空き容量 |

> ❌ **Intel Macに関する重要な注意**: 
> 
> このアプリケーションは Intelチップ搭載のMacコンピュータを**サポートしていません**。強制的に実行しても、以下の理由により体験は極めて貧弱になります：
> 1. **アーキテクチャの非互換性**: 内蔵のローカル推論エンジン (WebLLM/Ollama) は、Apple SiliconのARM64アーキテクチャとNPU/Metalハードウェアアクセラレーションに深く依存しています。
> 2. **ハードウェアアクセラレーションの欠如**: Intel Macにはユニファイドメモリアーキテクチャがありません。量子化モデルの実行は極めて遅く（1トークンの生成に数秒かかる場合がある）、デバイスの深刻な発熱を引き起こします。
> 3. **アーキテクチャのトレードオフ**: 最高の体験と最小のパッケージサイズを確保するため、x86_64アーキテクチャのサポートを削除しました。
> 
> Apple Silicon (Mシリーズ) チップを搭載したMacデバイスの使用を強く推奨します。

## 📦 インストール

### 🍎 macOS インストール

**方法 1: Mac App Store (推奨)**

App Storeで "WitNote" を検索するか、[ここをクリック](https://apps.apple.com/us/app/witnote-local-ai-writer/id6756833873?mt=12) してダウンロードおよびインストールしてください。自動更新が利用可能です。

**方法 2: DMG インストーラー**

1. `.dmg` ファイルをダウンロード
2. DMGをダブルクリックして開く
3. アプリをApplicationsフォルダにドラッグ

### 🪟 Windows インストール

1. `.exe` インストーラーをダウンロード
2. セットアップウィザードを実行
3. インストールパスを選択 (カスタマイズ可能)
4. インストール完了、デスクトップまたはスタートメニューから起動

### 🐧 Linux インストール

**AppImage (ユニバーサル):**
1. `.AppImage` ファイルをダウンロード
2. 右クリックのプロパティ -> プログラムとして実行を許可 (または `chmod +x WitNote*.AppImage`)
3. ダブルクリックして実行

**Deb (Ubuntu/Debian):**
1. `.deb` ファイルをダウンロード (例: `WitNote-1.3.3-amd64.deb`)
2. ターミナルでインストールコマンドを実行 (依存関係を自動処理):
   ```bash
   sudo apt install ./WitNote-1.3.3-amd64.deb
   ```

> 📝 **Windowsユーザーへの重要な注意**:
> 
> 高価なEVコード署名証明書を持たない個人開発者であるため、以下の状況に遭遇する可能性があります：
> 1. **SmartScreen**: 「WindowsによってPCが保護されました」(不明な発行元) と表示された場合、**「詳細情報」** -> **「実行」**をクリックしてください。
> 2. **ウイルス対策ソフトの警告**: Windows Defenderや他のウイルス対策ソフトがインストーラーをフラグ付けする可能性があります。プロジェクトはオープンソースで安全です。ブロックされた場合は、一時的にウイルス対策ソフトを無効にしてみてください。

## 🔧 AIエンジン情報

### 1. WebLLM (Light)
アプリには `qwen2.5:0.5b` モデルを搭載したWebLLMエンジンが内蔵されています (macOSのみ、WindowsユーザーはOllamaの使用を推奨)。
- **メリット**: 追加のソフトウェアインストール不要。初回のモデルダウンロード後は完全にオフラインで動作します。
- **最適**: クイックQ&A、シンプルなテキスト推敲、低スペックデバイス。

### 2. Ollama (Local Power)
ローカルで実行されているOllamaサービスへの接続をサポートします。
- **メリット**: 7B、14B、さらに大きなモデルを実行可能。強力なパフォーマンス、完全オフライン。
- **使用法**: まず [Ollama](https://ollama.com) をインストールし、設定で追加のモデル (例: qwen2.5:7b, llama3など) をダウンロードしてください。

### 3. Cloud API (Limitless Cloud)
OpenAI互換のCloud APIへの接続をサポートします。
- **メリット**: APIキーだけで地球上で最も強力なモデルにアクセスできます。
- **最適**: トップクラスの論理的推論、またはローカルハードウェアが大規模モデルをサポートできない場合。
- **設定**: 設定でAPI URLとキーを入力してください (OpenAI, Gemini, DeepSeek, Moonshotなどをサポート)。
