**智簡筆記本** 是一款本地優先的 AI 寫作伴侶，支持 macOS、Windows 和 Linux 平台。
支持 **Ollama / WebLLM / 雲端 API** 三種引擎自由切換，搭配極簡**原生卡片介面**，開箱即用。除模型下載外無雲端依賴，無隱私焦慮，讓智能回歸輕盈。

## 🌟 核心理念

- **智 (Smart)**: 三擎合一，自由選擇。
  - **WebLLM**: 輕量級模型，首次需聯網下載，之後完全離線可用。
  - **Ollama**: 本地強大模型，性能強勁，完全離線。
  - **Cloud API**: 連接雲端智慧，無限可能。
- **簡 (Simple)**: 拒絕繁雜。
  - iOS 風格卡片管理，拖曳即整理。
  - 智能專注模式，窗口變窄即刻變身純文本編輯器。
- **安 (Secure)**: 數據自治。
  - [**隱私政策**](PRIVACY.md)：100% 本地存儲，你的思想只屬於你。

## ✨ 功能亮點 (v1.3.3)

### 🆕 v1.3.3 新功能

- 🪟 **磨砂玻璃 UI** — 全新視覺設計，現代磨砂玻璃效果貫穿全域
- 📐 **靈活三欄佈局** — 側邊欄、編輯區、AI 面板三欄均可自由調整寬度和獨立關閉
- ✋ **雙欄拖曳編輯** — 雙欄預覽模式下，自由拖曳分割線調整編輯區與預覽區比例
- 📂 **Finder 風格文件樹** — 顏色標記、拖曳移動、懸停展開文件夾，右鍵菜單全面升級
- 📄 **多格式支持** — 新增 PDF、Word 文檔預覽，支持常見圖片格式查看
  - 可編輯：`.md` `.txt`
  - 只讀預覽：`.pdf` `.docx`
  - 圖片查看：`.jpg` `.png` `.gif` `.webp`
- 🚀 **快速啟動** — 優化啟動畫面，大幅減少白屏等待時間
- 🔒 **App Store 上架** — 完全符合 Apple 沙盒安全標準，已正式上架 Mac App Store

### 🔧 核心功能

- 📝 **純本地筆記** — 選擇任意文件夾作為筆記庫，支持 `.txt` 和 `.md`
- 🤖 **三擎合一** — 自由切換 **WebLLM** (輕量)、**Ollama** (本地強力) 或 **雲端 API** (自定連接)
- ✨ **智能續寫** — 三檔預設 (Lite/Standard/Full)，按 Tab 逐句接受建議
- 🎭 **豐富角色庫** — 內置 10+ 種精選角色提示詞 (寫作、翻譯、潤色等)，支持一鍵切換與自定義
- 🌍 **全球溝通** — **支持 8 種語言**: 
  - 简体中文, English, 繁體中文, 日本語, 한국어, Français, Deutsch, Español
  - 介面與 AI 回覆自動適配您的語言設置
- 🔒 **隱私優先** — 所有 AI 推理在本地完成 (使用本地引擎時)，無數據上傳，通過 Apple 官方公證
- 💬 **高度定制** — 可自由編輯系統提示詞，打造專屬 AI 助手
- 🎨 **多主題切換** — 淺色 / 深色 / 禪意茶色，深色模式全面優化
- 🗂️ **卡片網格視圖** — iOS 風格，拖曳排序，上下文菜單完善
- 🔍 **上下文感知** — AI 可直接讀取當前文章或文件夾內容
- 🎯 **專注模式** — 窗口變窄自動切換純文本編輯體驗

## 💻 設備要求

### 🍎 macOS

| 項目 | 最低配置 | 推薦配置 |
|------|----------|----------|
| 系統版本 | macOS 12.0+ | macOS 13.0+ |
| 晶片 | **不予支持 (Intel 晶片)** | **Apple Silicon (M1/M2/M3/M4/M5)** |
| 記憶體 | - | 16GB+ |
| 硬碟 | - | SSD, 4GB+ 可用空間 |

> ❌ **關於 Intel Mac 的重要說明**: 
> 
> 本應用 **不支持** Intel 晶片的 Mac 電腦。即便強制運行，也會因以下原因導致極差體驗：
> 1. **底層架構不兼容**: 內置的本地推理引擎 (WebLLM/Ollama) 深度依賴 Apple Silicon 的 ARM64 架構與 NPU/Metal 硬件加速。
> 2. **硬件加速缺失**: Intel Mac 缺乏統一記憶體架構，運行量化模型速度極慢（生成一個字可能需數秒），且會導致設備嚴重發熱。
> 3. **架構取捨**: 為了保證最佳體驗與包體積，我們移除了對 x86_64 架構的支持。
> 
> 強烈建議使用搭載 Apple Silicon (M系列) 晶片的 Mac 設備。

### 🪟 Windows

| 項目 | 最低配置 | 推薦配置 |
|------|----------|----------|
| 系統版本 | Windows 10 (64-bit) | Windows 11 |
| 處理器 | Intel Core i5 / AMD Ryzen 5 | Intel Core i7 / AMD Ryzen 7 |
| 記憶體 | 8GB | 16GB+ |
| 硬碟 | 2GB 可用空間 | SSD, 4GB+ 可用空間 |
| 顯卡 | 集成顯卡 | 支持 Vulkan 的獨立顯卡 |

> ⚠️ **注意**: Windows ARM64 設備 (如 Surface Pro X) 現已原生支持！

## 📦 安装說明

### 🍎 macOS 安裝

**方式一：Mac App Store (推薦)**

直接在 App Store 搜索 "WitNote" 或 [點擊此處](https://apps.apple.com/us/app/witnote-local-ai-writer/id6756833873?mt=12) 下載安裝，享受自動更新。

**方式二：DMG 安裝包**

1. 下載 `.dmg` 文件
2. 雙擊打開 DMG
3. 將應用拖入 Applications 文件夾
4. 從啟動台運行

> 🎉 **好消息！**
>
> 本應用已通過 **Apple 官方公證 (Notarization)**！不會再出現"無法驗證開發者"的提示了！

### 🪟 Windows 安裝

1. 下載 `.exe` 安裝程序
2. 雙擊運行安裝嚮導
3. 選擇安裝路徑（可自定義）
4. 完成安裝，從桌面或開始菜單啟動

### 🐧 Linux 安裝

**AppImage (通用):**
1. 下載 `.AppImage` 文件
2. 右鍵屬性 -> 允許作為程序執行 (或 `chmod +x WitNote*.AppImage`)
3. 雙擊運行

**Deb (Ubuntu/Debian):**
1. 下載 `.deb` 文件 (如 `WitNote-1.3.3-amd64.deb`)
2. 在終端運行安裝命令 (這將自動處理依賴):
   ```bash
   sudo apt install ./WitNote-1.3.3-amd64.deb
   ```

> 📝 **注意事項 (Windows 用戶必讀)**:
> 
> 由於個人開發者暫時無法申請昂貴的 EV 代碼簽名證書，安裝時可能會遇到以下提示，請按步驟操作：
> 1. **SmartScreen 攔截**：如果彈出"Windows 已保護你的電腦"（未知發布者），請點擊 **"更多信息"** -> **"仍要運行"**。
> 2. **殺毒軟件提示**：可能會被 Windows Defender 或 360 等誤報。請放心，項目完全開源且安全，如遇阻攔建議暫時關閉殺毒軟件。

## 🔧 AI 引擎說明

### 1. WebLLM (輕量)
應用內置 WebLLM 引擎，默認搭載 `qwen2.5:0.5b` 模型（macOS 專用，Windows 推薦使用 Ollama）。
- **優點**: 無需安裝任何額外軟件，首次自動下載模型後即可完全離線使用。
- **適用**: 快速問答、簡單文本潤色，低配置電腦首選。

### 2. Ollama (本地強力)
支持連接本地運行的 Ollama 服務。
- **優點**: 可運行 7B、14B 甚至更大的模型，性能強大，完全離線。
- **使用**: 需先安裝 [Ollama](https://ollama.com)，然後在設置中下載更多模型 (如 qwen2.5:7b, llama3等)。

### 3. Cloud API (雲端無限)
支持連接兼容 OpenAI 格式的雲端 API。
- **優點**: 只要有 API Key，即可使用地球上最強大的模型。
- **適用**: 需要頂級邏輯推理能力的場景，或者電腦配置較低無法運行大模型時。
- **配置**: 在設置中填入 API 地址和 Key 即可 (支持 OpenAI, Gemini, DeepSeek, Moonshot 等)。
